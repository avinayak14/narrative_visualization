An Inspection of Operating Automobiles in 2017

 

Author: Avi Nayak

Description of the Data

The data being used here is based on averaged attributes of 2017 automobile manufacturers. It contains the make, fuel type, engine cylinder count, average highway MPG, and average city MPG of several different types of cars.

The primary goal of this interactive slideshow visualization is to exhibit a presentation via a set of interactive webpages. It is conveyed in narrative format, using data to tell a story, and it invites viewers to explore further and serves as a medium for data journalism.

Description of the Visualization: This narrative visualization follows an interactive slideshow structure to guide users through various selected scenarios.
